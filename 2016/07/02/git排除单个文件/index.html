在一些场景下不同分支的一些文件不应该合并，例如测试环境和生产环境的数据库配置文件。这种情况下即使在.git/info/exclude配置了不追踪这些文件，merge的时候还是会被merge。一种很好的办法是在.git/info/attributes中配置不被merge的文件（不包含文件路径），例如：
