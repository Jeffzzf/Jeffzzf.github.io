求最长公共子序列

使用DP算法，i, j分别表示,s1和s2的下标，转移方程为:

  • dp[i][j] = 0; if(i == 0 || j == 0)
  • dp[i][j] = dp[i - 1][j - 1] + 1; if(s1[i] == s2[j])
  • dp[i][j] = max(dp[i - 1][j], dp[i][j - 1]); if(s1[i] != s2[j])

    • dp[i][j] = dp[i - 1][j - 1]; if(s1[i] == s2[j])
    • dp[i][j] = 0; if(s1[i] != s2[j])
